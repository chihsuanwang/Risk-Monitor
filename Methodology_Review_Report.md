# USDTWD 異常偵測方法論 — 嚴格檢視報告

> 審閱角度：資深量化外匯交易員 / 風險模型審查
> 日期：2026-03-30
> 評級：⚠️ 方向正確，但存在數個需要修正的結構性問題

---

## 執行摘要

你的研究方向是正確的，使用 NDF 市場信號來預判 USDTWD 波動，在概念上是有效的，且三指標同時觸發的邏輯有助於控制誤報。**但在統計嚴謹性、信號多樣性、以及方法論設計上，存在幾個可能讓整個研究失效的根本問題。** 以下按嚴重程度排序。

---

## 一、致命問題（必須修正，否則結論不可信）

### 問題 1：異常定義存在路徑盲點（Label Design Flaw）

**你的定義**：T+22 相較於 T 日升值超過 3%，即為異常。

**問題所在**：你只比較兩個時間點，完全忽略中間的路徑。

```
反例：
  T+10 : 台幣升值 4.5%（這是真正的大波動！）
  T+22 : 台幣回落，相對 T 僅升值 1.8%
  → 你的定義：「非異常」
  → 真實情況：發生了極大的波動，只是被你的定義漏掉了
```

**另一個反例**：
```
  T+1 ~ T+21 : 台幣每天小幅升值
  T+22 : 剛好累計升值 3.01%
  → 你的定義：「異常」
  → 真實情況：這只是正常的趨勢性升值，不是大波動
```

**建議修正**：
- 用觀察窗內的**最大升值幅度**（Max Appreciation over T to T+22）取代點對點比較
- 或加入**波動加速度**：22日內是否出現連續5日以上、每日升值超過0.2%的情況
- 或使用 **Rolling Z-score**：當前22日升值幅度是否超過歷史分布的第95百分位

---

### 問題 2：觀察窗設計存在嚴重的前視偏差（Look-ahead Bias）

**你的設計**：觀察期 T-22 到 T+11；異常發生在 T+22。

**問題所在**：

```
時間軸：
T-22 ────────── T ────────── T+11 ────── T+22
[  觀察期開始   ][  交易日   ][觀察期結束  ][異常判定]
```

問題在於：T 到 T+11 這段數據——**你在 T 日當天根本不知道 T+22 會是什麼結果**。

如果你用 T+1 到 T+11 的指標數據來判斷「T+22 是否異常」，那麼：
- 你的指標實際上是在**異常發生的過程中**採集的
- 這會讓本來沒有預測力的指標，看起來有很強的信號
- 這是回測中最常見、最隱蔽的錯誤之一

**建議修正**：
- **預測框架**：只用 T-22 到 T 的數據預測 T+22 的結果
- **確認框架**（如果用意是監控，而非預測）：明確區分「觸發條件」與「確認條件」，並在報告中標示清楚
- 整個觀察窗應該重新定義為：用 T 日及以前的數據，預測未來 N 天的最大升值幅度

---

### 問題 3：三個指標本質上是同一類信號（Signal Diversification Failure）

**你的三個指標**：
1. 1M NDF 年化成本 - 12M NDF 年化成本 > 5%
2. 1M NDF 年化成本 > 8%
3. CVIX > 一年移動平均 + 2 倍標準差

**問題所在**：三個指標全部來自**同一個市場（外匯衍生品市場）**，且高度相關。

```
當1M NDF成本高漲時：
  → 1M NDF年化成本 > 8%  → 觸發
  → 1M-12M利差擴大       → 觸發（因為1M漲但12M未必同等漲）
  → 外匯隱含波動率上升    → CVIX 往往也上升 → 觸發
```

這三個指標本質上在衡量同一件事：**短端外匯市場的壓力**。

**AND 邏輯的誤解**：你以為三個指標同時觸發代表「三個獨立信號同時確認」，實際上可能只是「一個市場信號，從三個角度量測到了」。

**更嚴重的問題**：你研究了 5 大類指標（NDF、美元波動、競爭對手匯率、資金面、避險成本），最終三個指標全部來自前兩類，這表示你可能**過度擬合**到某一類特徵，並丟棄了真正有互補性的信號。

**建議修正**：
- 計算三個指標之間的條件相關係數——若相關係數 > 0.7，三合一的 AND 邏輯只相當於一個獨立信號
- 重新評估外資買賣超、台股信號的預測力，這些是**真正獨立的信號來源**
- 考慮用 **Ensemble 或 Voting 架構**替代純 AND 邏輯

## 二、重要問題（顯著影響結論可靠性）

### 問題 4：重疊樣本造成的統計失效（Overlapping Observations）

你用 22 日窗口在 20 年日資料上滾動計算，這代表：

- **相鄰觀測樣本高度重疊**（連續兩天的觀察窗共享 21 天的數據）
- 你的樣本實際上不是「獨立同分布（i.i.d.）」
- 標準的統計顯著性測試（t-test、p-value）在此框架下**嚴重高估統計顯著性**

```
舉例：
  20年 × 250交易日 = 5000個觀測值
  但 22日窗口重疊後，獨立樣本數 ≈ 5000 / 22 ≈ 227個

  你以為有 5000 個樣本的統計信心，實際上只有 ~227 個
```

**建議修正**：
- 使用 **Block Bootstrap** 方法估計信心區間，而非標準正態假設
- 僅使用非重疊窗口進行初步統計驗證（每22天取一個樣本 = 約227個獨立事件）
- 使用 **Hansen-Hodrick 標準誤** 校正時序相關性

---

### 問題 5：閾值的設定缺乏統計依據

**你的閾值**：
- 1M - 12M > 5%
- 1M > 8%
- CVIX > MA(252) + 2σ

**問題所在**：這些數字是怎麼來的？是直覺、是觀察歷史後的事後挑選，還是有嚴謹的最優化過程？

如果是「看了歷史數據後覺得 5% 這條線比較準」，那就是**過度擬合（Overfitting）**。

CVIX 的 2 倍標準差在統計上對應約 95% 分位數，這個邏輯合理。但 5% 和 8% 是否也對應到合理的歷史分位數？

**建議修正**：
- 對每個指標進行 **Grid Search**：枚舉不同閾值，計算對應的 Hit Rate、False Positive Rate，繪製 ROC 曲線
- 選擇閾值的依據應是：「最大化 Hit Rate，同時控制 False Positive Rate 在可接受範圍」
- 使用 **Walk-forward Validation**：前15年訓練（找最優閾值），後5年驗證（不再調整閾值）

---

### 問題 6：未處理 CBC 干預的制度性影響

你的 20 年資料橫跨了多個 CBC 行長任期，而 CBC 的干預風格因人而異。

- 2000~2018 年彭淮南時代：積極干預，台幣升值空間受壓制
- 2018年後楊金龍時代：略微放寬，但仍積極

**問題**：你的「異常升值3%」在干預積極的時期幾乎不可能發生，但在某些特定事件（如彭時代末期的幾次破防）卻確實出現。

**這意味著**：
- 在彭淮南時代，你的指標可能顯示「預測力很低」，但這不是指標的問題，而是 CBC 壓制了結果
- 你的指標實際上預測的是「市場升值壓力是否足夠大，連 CBC 都擋不住」
- 這個闡釋比「預測 USDTWD 升值」更準確，也更有意義

**建議修正**：
- 在回測中加入 **CBC 干預強度指標**（如：USDTWD 距近3年低點的距離）作為調整因子
- 分析：「指標觸發時，如果沒有 CBC 干預，理論升值幅度應為多少？」
- 將樣本期依干預風格分段分析

## 三、方法論改善建議

### 問題 7：研究設計缺乏 Out-of-Sample 驗證

你用 20 年數據找到這三個指標，這 20 年的數據**是否也是你用來選擇指標的依據**？

如果是，那你做的是「**在全樣本上選最優指標，再在全樣本上驗證**」，這在統計上等同於沒有驗證。

**建議做法**：
```
正確的驗證流程：

訓練期（2003~2018）：
  ├─ 探索候選指標
  ├─ 優化閾值
  └─ 確定三指標AND邏輯

驗證期（2019~2024）：
  ├─ 閾值完全不再調整
  ├─ 計算指標在從未見過的數據上的表現
  └─ 若表現顯著衰退 → 方法存在過擬合
```

2019年後剛好涵蓋：COVID（2020）、Fed 升息（2022）、AI 熱潮（2024），是理想的壓力測試期。

---

### 問題 8：信號觸發的時序未被研究

你的 AND 邏輯要求三個指標**全部觸發**，但你是否研究過：

- 三個指標通常哪個先觸發？
- 「第一個指標觸發」到「第三個指標觸發」的平均間隔是幾天？
- 是否存在一個「第一觸發 → 第二觸發 → 第三觸發 → N天後大升值」的序列模式？

如果三個指標通常在 3 天內同時觸發，AND 邏輯是有效的。
但如果間隔往往是 2~3 週，那麼等待全部觸發可能讓你**錯失最佳預警時間點**。

**建議**：繪製三個指標各自觸發後，到大升值事件的時間分布直方圖。

---

### 問題 9：僅選用最終三指標，丟棄的指標未被充分評估

你說你研究了5大類指標，最後只留下3個，全都來自 NDF/Vol 類。

這讓我擔心的是：

**資金面指標（外資買超、台股）的預測力是否被不公平地低估了？**

NDF 指標數據更完整（日資料，無缺值），而外資買超數據可能有週末空缺、修正版本等問題。如果你的評估框架對數據品質較差的指標有懲罰，那麼「NDF 最優」的結論可能只是數據品質差異的反映，而非真實預測力的差異。

**建議**：
- 公開各類指標的評估結果（Hit Rate、False Positive Rate），而非只保留最終三個
- 即使外資買超的獨立預測力較弱，考慮它作為**確認指標**的角色（在主信號觸發後，外資買超同向 → 信號升級）

## 四、正面肯定：做得好的地方

在批評之餘，也必須指出這個研究設計的亮點：

| 優點 | 說明 |
|------|------|
| **選用NDF市場信號** | NDF 不受 CBC 管制，反映真實市場預期，是正確的信號來源選擇 |
| **使用年化成本而非絕對點數** | 年化處理讓不同時期的數據具有可比性，統計學上正確 |
| **多指標AND邏輯** | 有效降低誤報率，符合實際風控操作需求 |
| **20年數據** | 覆蓋多個經濟周期，有足夠的事件樣本 |
| **1M-12M 期限利差** | 這是個精妙的選擇——短端壓力大於長端，代表市場在押注即將發生的短期升值 |
| **CVIX 的動態閾值** | 用移動平均+標準差而非固定值，能適應市場波動率的結構性變化 |
