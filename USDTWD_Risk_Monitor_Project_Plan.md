## 1. 專案背景與目標

### 1.1 使用者背景
- 職位：外匯風險管理人員
- 核心需求：提前掌握 USDTWD 重大波動訊號，尤其是**台幣升值**方向
- 工作頻率：每週一次定期審視

### 1.2 核心問題定義

> **我們想回答的問題是：未來 4 週內，USDTWD 是否有可能發生超出正常波動範圍的大幅移動？方向尤其關注台幣升值（USDTWD 下跌）。**

### 1.3 成功標準
- 在重大波動發生前 **2~4 週**能夠發出警示
- 誤報率（False Positive）控制在合理範圍，避免警報疲勞
- 系統能透過歷史回測驗證有效性

## 2. USDTWD 市場特性深度分析

### 2.1 USDTWD 不是一個「正常」的匯率

多數外匯教科書的框架不能直接套用在 USDTWD，原因如下：

| 特性 | 說明 |
|------|------|
| **管制型市場** | USDTWD 是管制貨幣，不像 EURUSD 完全自由浮動 |
| **央行強力干預** | 台灣央行（CBC）是全球最積極干預的央行之一，單日波動常被人為壓制 |
| **流動性集中** | 主要流動性在台北時間上午，收盤前15分鐘常有央行出手痕跡 |
| **離岸NDF存在** | 海外機構透過NDF交易，形成與在岸市場的價差，反映真實市場預期 |
| **季節性明顯** | 年底外資獲利了結、農曆年前企業換匯需求，造成周期性波動 |

### 2.2 USDTWD 的正常波動範圍

根據近10年歷史數據：

| 時間維度 | 正常波動幅度 | 大波動定義 |
|---------|------------|----------|
| 單日 | ±0.1% ~ ±0.3% | 超過 ±0.5% |
| 單週 | ±0.3% ~ ±0.8% | 超過 ±1.5% |
| 單月 | ±1% ~ ±2% | 超過 ±3% |

> **風控人員注意**：「大波動」的定義應根據您公司的風險敞口規模自行校準。本報告以市場慣例為準。

### 2.3 USDTWD 與全球風險情緒的關係

台幣是**風險貨幣（Risk Currency）**：
- **全球風險偏好上升**（Risk-on）→ 外資流入亞洲股市 → 台幣升值
- **全球風險趨避**（Risk-off）→ 外資撤出 → 台幣貶值、USD走強

這個框架在 2008、2020、2022 年都得到充分驗證。

## 3. 台幣升貶值的核心驅動機制

### 3.1 台幣升值的三大引擎

#### 引擎一：外資買超台股（最即時、最直接）

```
外資看好台灣科技股
    ↓
買入台股需要先將 USD 換成 TWD
    ↓
USD 供給增加 → TWD 需求增加
    ↓
USDTWD 下跌（台幣升值）
```

**關鍵數字**：外資單週若淨買超超過 500 億台幣，對匯率有顯著推升效果。

#### 引擎二：出口商結匯壓力（有時間差）

```
台灣半導體 / 電子業接到大訂單（月外銷訂單公布）
    ↓
1~2 個月後，貨物出口，收到 USD 貨款
    ↓
出口商需將 USD 換成 TWD 支付員工薪資、原料成本
    ↓
持續性結匯壓力 → 台幣升值
```

**注意**：出口商通常會等待有利匯率再結匯，這個行為本身就會造成「壓縮後的爆發式升值」。

#### 引擎三：美元全球走弱（系統性因素）

```
Fed 降息預期升溫 / 美國經濟數據轉弱
    ↓
DXY（美元指數）走弱
    ↓
全球新興市場貨幣同步升值
    ↓
台幣跟漲，且因結構性因素往往升幅超過同儕
```

### 3.2 台幣貶值的觸發因素（同樣重要）

了解反方向的力量，才能判斷升值訊號是否可持續：

| 貶值壓力來源 | 說明 | 頻率 |
|------------|------|-----|
| **Fed 升息 / 鷹派轉向** | USD 吸引力上升，資金回流美國 | 周期性 |
| **地緣政治風險（台海）** | 台灣特有的地緣溢價，升溫時外資撤退 | 偶發 |
| **TSMC 等大廠海外匯款** | 赴美建廠需匯出大量 USD，形成買匯壓力 | 重大事件 |
| **年底外資獲利了結** | 12月外資賣台股換回USD，季節性貶值 | 年度規律 |
| **進口商避險買匯** | 台灣石油全靠進口，油價大漲時集中買USD | 偶發 |

## 4. 先行指標選擇與分析

以下為本系統擬追蹤的核心先行指標，依領先時間排序：

### 4.1 指標一：台灣月外銷訂單（Ministry of Economic Affairs）

| 屬性 | 內容 |
|------|------|
| **領先時間** | 1~2 個月 |
| **公布頻率** | 每月約 25 日 |
| **數據來源** | 經濟部統計處 |
| **核心邏輯** | 訂單 → 生產 → 出口 → 結匯，是資金流最上游的指標 |
| **警示訊號** | 月增率超過 +10%，或連續3個月正成長 |
| **反向警示** | 連續2個月衰退，升值壓力將減弱 |

> **資深交易員提醒**：外銷訂單中，**資訊通信產品（ICT）佔比最高**，主要反映半導體與電子產品需求，這部分的變化對 USDTWD 影響最大。

### 4.2 指標二：費城半導體指數（SOX）

| 屬性 | 內容 |
|------|------|
| **領先時間** | 2~4 週 |
| **公布頻率** | 每日（即時） |
| **數據來源** | Yahoo Finance（免費） |
| **核心邏輯** | SOX 上漲 → 全球晶片需求旺 → 台灣訂單增加 → 未來結匯壓力大 |
| **警示訊號** | SOX 突破近3個月高點，且週漲幅超過 +5% |
| **分析重點** | SOX 與 USDTWD 的20週滾動相關係數，相關係數高時訊號更可靠 |

### 4.3 指標三：外資在台股累計淨買超

| 屬性 | 內容 |
|------|------|
| **領先時間** | 即時 ~ 2 週（累積效應） |
| **公布頻率** | 每日收盤後公布 |
| **數據來源** | 台灣證券交易所（TWSE） |
| **核心邏輯** | 外資買股同時換匯，累積量代表持續性換匯壓力 |
| **警示訊號** | 連續4週淨買超，且累計超過 2,000 億台幣 |
| **注意事項** | 須區分「股票買賣」與「ETF申贖」，後者對匯率影響較小 |

### 4.4 指標四：美元指數（DXY）

| 屬性 | 內容 |
|------|------|
| **領先時間** | 1~3 週 |
| **公布頻率** | 每日（即時） |
| **數據來源** | Yahoo Finance（免費） |
| **核心邏輯** | DXY 為全球 USD 強弱的綜合指標，台幣與 DXY 高度負相關 |
| **警示訊號** | DXY 跌破20日移動平均線，且 RSI < 45 |
| **分析重點** | DXY 週線若形成頭部型態（如雙頂），通常預示 USD 中期走弱 |

### 4.5 指標五：美台2年期公債利差

| 屬性 | 內容 |
|------|------|
| **領先時間** | 2~6 週 |
| **公布頻率** | 每日 |
| **數據來源** | FRED（美國）/ CBC 官網（台灣） |
| **核心邏輯** | 利差縮小 → 持有 USD 的利息優勢下降 → 資金轉向 TWD |
| **警示訊號** | 2年期利差連續4週收窄，且當前利差低於近12個月平均 |
| **延伸分析** | Fed 利率點陣圖（Dot Plot）變化是最重要的前瞻指引 |

### 4.6 指標六：VIX 恐慌指數

| 屬性 | 內容 |
|------|------|
| **領先時間** | 即時（確認指標，非領先指標） |
| **公布頻率** | 每日（即時） |
| **數據來源** | Yahoo Finance（免費） |
| **核心邏輯** | VIX < 15 = 低恐慌，有利風險貨幣（台幣升值環境）；VIX > 25 = 風險趨避警示 |
| **使用方式** | 作為其他指標的過濾器：當 VIX 偏低時，升值訊號更可信 |
| **重要警示** | VIX 從低位急速跳升（單週 +50%）是大波動的最即時警報 |

### 4.7 指標綜合評分矩陣

系統將依以下邏輯產生綜合警示分數（0~100分）：

```
升值壓力分數 =
    外銷訂單信號 × 25% +
    SOX 信號     × 20% +
    外資買超信號  × 25% +
    DXY 信號     × 20% +
    利差信號      × 10%

VIX 調整：若 VIX > 20，升值壓力分數打折 × 0.7
          若 VIX < 15，升值壓力分數加乘 × 1.1

最終警示等級：
    分數 > 70 → 🔴 高度升值風險警示
    分數 50~70 → 🟡 中度升值壓力觀察
    分數 < 50 → 🟢 正常範圍
```

## 5. 風險警告：資深交易員視角

### ⚠️ 警告一：台灣央行（CBC）是最大的不可測黑盒子

**這是最重要的市場結構風險，任何模型都無法完全預測。**

CBC 的干預邏輯：
- 台幣**單日升值幅度**超過約 0.3% 時，CBC 傾向進場調節
- CBC 特別在意年末收盤價（影響公司財報）
- 干預手段：直接在外匯市場買入 USD，或透過指定銀行操作
- CBC **絕不公布**干預時點與規模（與多數央行不同）

**對你的影響**：即使所有先行指標都指向台幣大幅升值，CBC 的存在意味著升值往往是「緩慢擠壓式」而非「快速突破式」。短期的重大波動更可能發生在**貶值**方向（因 CBC 不會阻止貶值）。

**建議**：系統中加入「CBC 干預風險等級」評估模組，依據近期 USDTWD 距離歷史低點的距離來判斷央行干預可能性。

---

### ⚠️ 警告二：TSMC 海外建廠的「反直覺效應」

```
表面邏輯：TSMC 訂單大好 → 台灣出口增加 → 台幣升值
真實情況：TSMC 宣布美國/日本建廠 → 大規模匯款海外 → 台灣 USD 流出 → 台幣貶值
```

近年案例：
- 2022年 TSMC 宣布亞利桑那州擴廠，匯款壓力抵消了部分出口升值壓力
- 2024年 熊本廠開幕期間，相關資金流出同樣造成短期貶值壓力

**建議**：系統中加入「TSMC 重大資本支出事件」人工標記功能，避免模型在此期間發出誤導訊號。

---

### ⚠️ 警告三：NDF 離岸市場是「真實意圖」的顯示器

在岸 USDTWD 現貨受 CBC 管制，但**離岸 NDF（不可交割遠期）市場不受管制**，反映的是全球市場對台幣的真實預期。

- **NDF 貼水**（NDF < 現貨）：市場預期台幣升值
- **NDF 溢價**（NDF > 現貨）：市場預期台幣貶值
- 1個月期 NDF 與現貨的價差，通常比現貨本身更先反映趨勢轉折

**建議**：若未來有 Bloomberg 授權，NDF 報價應列為**第一優先**監控指標。免費 API 目前無法取得此數據，這是一個重要的數據缺口。

---

### ⚠️ 警告四：地緣政治尾部風險（台灣特有）

台灣面對的地緣政治風險是全球外匯市場中**最具不對稱性**的一種：
- 日常情況下，地緣政治溢價很小（市場已定價）
- 但一旦情勢升溫，USDTWD 的波動幅度可能超出所有歷史模型的預測範圍

**建議**：系統中加入「地緣政治新聞關鍵字監控」，或設立人工輸入的「地緣政治風險等級」欄位（1~5 分），影響最終風險評分。

---

### ⚠️ 警告五：相關性在壓力時期會崩潰

在正常市場環境下，本系統的先行指標與 USDTWD 的相關性良好。但**在市場壓力時期**（例如 COVID、金融危機），所有資產相關性趨向 1，指標失效，只剩 VIX 有意義。

這意味著：
- 本系統適合預警「可預期的趨勢性波動」
- 對於「突發性黑天鵝事件」，任何量化模型都無能為力
- 風險管理人員必須在系統之外保持對尾部風險的警惕

## 8. 手動上傳工作流程設計

## 11. 回測框架設計

### 11.1 回測目標

驗證各先行指標在以下歷史事件前的信號有效性：

| 事件 | 日期 | USDTWD 波動 | 方向 |
|------|------|------------|------|
| COVID-19 爆發 | 2020-03 | 大幅波動 | 先升後急貶 |
| 全球股市復甦 | 2020-Q3~Q4 | 台幣大升值 | 升值 |
| Fed 開始升息 | 2022-03 | 台幣走貶 | 貶值 |
| Fed 升息放緩預期 | 2023-H2 | 台幣反彈 | 升值 |
| AI 科技股熱潮 | 2024 | 外資大量流入 | 升值 |

### 11.2 回測評估指標

- **Lead Time**：指標發出信號到 USDTWD 大波動的平均天數
- **Hit Rate**：信號準確率（發出升值警示後確實升值的比例）
- **False Positive Rate**：誤報率
- **Miss Rate**：漏報率（大波動前未發出信號的比例）
- **Sharpe-like Ratio**：信號的風險調整收益率（用於評估指標組合優化）

### 11.3 回測所需最少數據量

| 指標 | 所需最短歷史 | 優先取得範圍 |
|------|-----------|------------|
| USDTWD | 5年 | 2019~至今 |
| SOX, DXY, VIX | 5年 | 2019~至今 |
| 外銷訂單 | 5年 | 2019~至今 |
| 外資買超 | 3年 | 2022~至今 |

## 12. 歷史重大事件回顧

### 12.1 2020年：COVID引發的最大單月波動

| 時期 | USDTWD 走勢 | 先行指標表現 |
|------|------------|------------|
| 2020-01~02 | 平穩 | VIX 開始上升但不明顯 |
| 2020-03 | 急貶（TWD 大貶）| VIX 急升至 80，SOX 崩跌，外資大量賣超 |
| 2020-04~12 | 持續大幅升值 | SOX 強勁回升，外資持續買超，DXY 走弱 |

**結論**：2020年的升值是「危機後復甦型」，SOX 和外資買超是最有效的領先指標。

### 12.2 2022年：Fed 升息週期引發的台幣貶值

| 時期 | USDTWD 走勢 | 先行指標表現 |
|------|------------|------------|
| 2022-01 | USDTWD 開始走升（TWD貶） | 美台利差擴大，DXY 走強 |
| 2022-06 | USDTWD 到達波段高點 | VIX 高位，外資大量賣超台股 |
| 2022-Q4 | 開始回落 | Fed 升息放緩預期，SOX 反彈 |

**結論**：2022年的貶值由利率差驅動，美台利差是最有效的領先指標（領先約 6~8 週）。

### 12.3 2024年：AI 科技股熱潮與台幣升值

- TSMC 業績大幅超預期 → 外資大買台股 → 台幣持續升值壓力
- CBC 多次干預，但市場升值趨勢未改
- 此階段 SOX + 外資買超 組合信號最強

## 14. 待討論與決策事項

### 14.1 部署方式
- **本機執行**（最簡單，僅自己使用）：Flask 本地 server

**建議**：先做本機版本，確認功能後再考慮升級。

- 申請網址：https://fred.stlouisfed.org/docs/api/api_key.html

- TWSE 每日公布，但您的需求是每週一次

## 附錄 B：名詞解釋

| 名詞 | 解釋 |
|------|------|
| **USDTWD** | 美元兌台幣匯率，數值越大代表台幣越弱（貶值） |
| **NDF** | 不可交割遠期外匯，在離岸市場交易，不涉及實物交割 |
| **CBC** | 中央銀行（中華民國），即台灣央行 |
| **SOX** | 費城半導體指數（Philadelphia Semiconductor Index） |
| **DXY** | 美元指數，衡量美元對一籃子主要貨幣的強弱 |
| **VIX** | 芝加哥期權交易所波動率指數，俗稱「恐慌指數」 |
| **Z-score** | 標準化分數，衡量當前值偏離歷史均值幾個標準差 |
| **Risk-on / Risk-off** | 市場風險偏好開啟/關閉，影響資金在全球的分配 |
